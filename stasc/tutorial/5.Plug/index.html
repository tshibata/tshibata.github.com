5. Plug

You can't change value of stasc bond. If you need to change, you have to use stasc plug like Table.ss.xml. Inside the stasc plug tag, there must be only one element tree. And the element tree must not have any stasc tag in it because it can be replaced. The element tree is just provisional. In airhockey.js, initially title of the game is shown. But it is replaced with Clear.ss.xml and Score.ss.xml alternatively. And finally it is replaced with End.ss.xml.

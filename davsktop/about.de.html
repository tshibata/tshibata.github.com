Tipps und Tricks

  • Sie können Datei umbenennen durch Ziehen und Ablegen in die gleich Verzeichnis. Davsktop fragt Ihnen neues Name wenn die Verzeichnis hat schon Datei mit gleich Name.
  • Link icon() ist nur ein hyper link zu Datei. Mit es können Sie alles tun Sie können mit Ihr Webbrowser, z.b. laden die Datei.

Einstellungen

Sie können stellen Davsktop ein durch schreiben in config.js Variablen.
Name Funktion Voreinstellung
BACKGROUND_IMAGE Grundbild undefined
MONTHS Namen von of Monaten ["Jan", "Feb", "Mar", "Apr", "May", "Jun", "Jul", "Aug", "Sep", "Oct", "Nov", "Dec"]
TEXT_MIME_PATTERN Muster für MIME-Type von Textdatei /^text\//
TEXT_PATH_PATTERN Muster für Pfad von Textdatei /\.txt$/

Bekkante Probleme

  • Mit Chrome und IE10, können Sie Datei kopien nicht. Statt wegen Sie immer.
